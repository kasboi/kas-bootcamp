## Signup Form

Create a signup form to gather user information. Expected inputs:
1. First name
2. Last name
3. E-mail
4. Password
5. Date of birth
6. Submit button
7. Reset button
8. Dropdown (Select favourite something)

Do your research on the appropriate input type to use for each of these inputs.

## State, Capitals & Motto

Use the HTML `<table>` tag to list at least 5 states in Nigeria with their capital and motto

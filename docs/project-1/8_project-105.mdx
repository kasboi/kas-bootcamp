First thing we did was move the section out of the `main` element because the padding on the `main` affects the entire section but we want the background image to span the entire width of the container.

```css {4-6}
/* SECTION - IMAGES */
section.images {
  text-align: center;
  padding: 4rem 0;
  background-attachment: fixed;
  background-image: url("https://i.ibb.co/NZYDg9K/ben10-background.png");
}
```

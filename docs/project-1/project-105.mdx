## `background`

Let's talk about CSS `background` property before moving on with our mini project. The `background` property is a shorthand for many background properties.

## `background-color`

This we have used multiple times for defining the background colour of an element.

```css
p {
  background-color: #f3f;
}

section {
  /* This will also work */
  background: rebeccapurple;
}
```

## `background-image`

This is for setting an image as the background of an element. By default, the image is repeated to fill the entire container vertically and horizontally.

In the example below we have a container which has a width of `1200px` and height of `600px`. The background image applied to it has a width of `556px` and height of `371px` and you can see it being repeated to fill the entire `div` container.

### `linear-gradient()`

It's not just images you can specify for the `background-image` property, you can also make use of linear-gradient function which allows us to blend multiple colours together.

To use it, you must define at least two colours, the starting colour to transition from and the ending colour. You can also
set a direction (or angle).

## `background-repeat`

This property is used to specify how the image should be repeated, by default it is repeated vertically and horizontally. You can change this to:

- `no-repeat`: The image does not repeat in either axis
- `repeat-x`: To repeat in the vertical direction only
- `repeat-y`: To repeat in the horizontal direction only

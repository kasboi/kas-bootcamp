We have a problem. The fixed `header` is nowhere to be found and the reason is simple, both the `header` and the `.banner` section are positioned and occupy overlaping space. Currently, the banner section is placed on top of the header, the solution is to specify which positioned element would be placed on the other and the property for that is `z-index`

```css showLineNumbers {10}
/* CSS styles */
header {
  color: #fff;
  background-color: #728308;
  padding: 2rem 0;
  text-align: center;
  /*  fixed positioning  */
  position: fixed;
  top: 0;
  width: 100%;
  z-index: 10;
}
/* CSS styles */
```

By giving the `header` a higher z-index, it is placed on the `.banner` section.

### Pseudo-classes

A pseudo-class targets and styles an element in a specific state.

For instance, it can:

- Change an element's appearance when the user hovers over it with a mouse
- Differentiate styles for links that have been visited versus those that haven't

```css
nav li a:hover {
  text-decoration: underline;
}
nav li a.join {
  color: #728308;
}
```

- ` nav li a:hover`: This targets any `<a>` (link) element within a `<li>` list item, specifically when a user hovers over it with their mouse.

  - `text-decoration: underline;`: When the user hovers over the link, this rule adds an underline, visually indicating the hover effect for enhanced interaction.

- `nav li a.join`: This targets the `<a>` link element with the class join inside a `<li>` element within a `<nav>` section.
  - `color: #728308;`: The color for this link is set to `#728308`, a shade of green, giving it a distinct appearance from other links.

### Pseudo-elements

A CSS pseudo-element allows you to style particular parts of an element.

For instance, it can:

- Style the first letter or first line of text within an element
- Add content before or after an element

```css
article p::first-line {
  font-weight: 600;
  font-size: 2rem;
}

article p::selection {
  background: #728308;
  color: #fff;
}
```

- `p::first-line`: The `::first-line` pseudo-element targets only the first line of each `<p>` element within an `<article>`.
- `p::selection`: The `::selection` pseudo-element styles the appearance of text when a user selects it (highlights it with a cursor).
- `background: #728308;`: Sets the background color of selected text to a green shade.
- `color: #fff;`: Changes the text color to white when selected, ensuring high contrast for readability against the green background.

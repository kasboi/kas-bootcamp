## Section Images, CSS `transform` & `transition`

```html
<main>
  <!-- prev. code -->
  <!--  Images Section  -->
  <section class="images">
    <img src="./img/accelerator.png" alt="ben10 accelerator" />
    <img src="./img/diamond.png" alt="ben10 diamond" />
    <img src="./img/four_arms.png" alt="ben10 four arms" />
  </section>
</main>
```

**CSS**

```css
section.images {
  text-align: center;
}

section.images img {
  width: 32rem;
  margin: 0 1rem;
  vertical-align: middle;
}
```

- `text-align: center;`: This sets the layout of the .images section to a flex container, allowing its child elements to be aligned and spaced using flexbox properties. Images are `inline` by default so it flows with text, this allows the text alignment to work.

### CSS `transform`

The CSS `transform` property allows elements to be visually altered in two-dimensional or three-dimensional space without affecting the document flow. It can be used to rotate, scale, skew, or translate elements, creating dynamic visual effects.

- `rotate()`: Rotates an element by a specified angle.
- `scale()`: Changes the element's size.
- `skew()`: Tilts the element along the x or y axis.
- `translate()`: Moves the element from its original position.

### CSS `transition`

The CSS `transition` property allows for smooth changes between property values over a specified duration, making animations more fluid and visually appealing. It's commonly used to animate elements on events like `hover`, `focus`.

### Key Properties:

- `transition-property`: Specifies the CSS property to transition (e.g., color, width, transform).
- `transition-duration`: Defines how long the transition lasts (e.g., 0.5s, 2s).
- `transition-timing-function`: Controls the speed of the transition over time (e.g., ease, linear, ease-in, ease-out).
- `transition-delay`: Sets a delay before the transition starts (e.g., 1s).

Two things must be specified when using the transition property:

1. CSS property the effect is to be added to
2. Duration of the effect

**Updated CSS**

```css
/* SECTION - IMAGES */
section.images {
  text-align: center;
}

section.images img {
  width: 32rem;
  margin: 0 1rem;
  vertical-align: middle;
  transition: all 0.2s ease-in-out;
}

section.images img:hover {
  transform: scale(1.25);
}
```
